Cobblemon Repel NeoReforged is addon for [Cobblemon](https://www.example.com) that adds 3 new blocks that prevent Pokemon spawns in configurable radius. This mod is client-server side!

### Configuration
---
The range of each Repel can be configured in `config/repelreforged-common` betwen 0 to 512 blocks. <br><br> By default:<br> 
- **Regular Repel** works on **48** blocks (**3x3** chunks) <br> 
- **Super Repel** on **96** blocks (**6x6 **chunks) <br> 
- **Max Repel** on **144** blocks (**9x9** chunks). <br>

### Mod Support
---
Cobblemon Repel has support for [Cobblemon Spawn Notification](https://modrinth.com/mod/cobblemon-spawn-notification), where it disables the broadcast if the spawned Pokémon was caught by a Repel.
